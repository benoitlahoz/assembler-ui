# Installation

Get started with AssemblerUI by setting up the foundation in your project. Choose the method that works best for your workflow.

## Prerequisites

Before installing AssemblerUI, ensure you have the following:

- **Node.js** 18+ or 20+
- **Vue 3.3+** or **Nuxt 3.8+**
- **Tailwind CSS** configured in your project

## Quick Start

AssemblerUI follows the **copy & paste** philosophy. You don't install a package; instead, you copy the components you need directly into your project.

::steps

### Install Tailwind CSS

If you haven't already, install and configure Tailwind CSS in your project.

::code-group

```bash [npm]
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

```bash [yarn]
yarn add -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

```bash [pnpm]
pnpm add -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

::

### Configure Tailwind CSS

Add the following to your `tailwind.config.js`:

```js [tailwind.config.js]
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./components/**/*.{js,vue,ts}",
    "./layouts/**/*.vue",
    "./pages/**/*.vue",
    "./plugins/**/*.{js,ts}",
    "./app.vue",
    "./error.vue"
  ],
  theme: {
    extend: {
      colors: {
        border: "hsl(var(--border))",
        input: "hsl(var(--input))",
        ring: "hsl(var(--ring))",
        background: "hsl(var(--background))",
        foreground: "hsl(var(--foreground))",
        primary: {
          DEFAULT: "hsl(var(--primary))",
          foreground: "hsl(var(--primary-foreground))",
        },
        secondary: {
          DEFAULT: "hsl(var(--secondary))",
          foreground: "hsl(var(--secondary-foreground))",
        },
        destructive: {
          DEFAULT: "hsl(var(--destructive))",
          foreground: "hsl(var(--destructive-foreground))",
        },
        muted: {
          DEFAULT: "hsl(var(--muted))",
          foreground: "hsl(var(--muted-foreground))",
        },
        accent: {
          DEFAULT: "hsl(var(--accent))",
          foreground: "hsl(var(--accent-foreground))",
        },
        popover: {
          DEFAULT: "hsl(var(--popover))",
          foreground: "hsl(var(--popover-foreground))",
        },
        card: {
          DEFAULT: "hsl(var(--card))",
          foreground: "hsl(var(--card-foreground))",
        },
      },
      borderRadius: {
        lg: "var(--radius)",
        md: "calc(var(--radius) - 2px)",
        sm: "calc(var(--radius) - 4px)",
      },
    },
  },
  plugins: [],
}
```

### Install Required Dependencies

Some components require additional dependencies. Install them as needed:

::code-group

```bash [npm]
npm install clsx tailwind-merge class-variance-authority
```

```bash [yarn]
yarn add clsx tailwind-merge class-variance-authority
```

```bash [pnpm]
pnpm add clsx tailwind-merge class-variance-authority
```

::

### Create Utility Functions

Create a utility file for common functions:

```ts [lib/utils.ts]
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

::

## Framework-Specific Setup

### Nuxt 3

If you're using Nuxt 3, add the CSS file to your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  css: ['~/assets/css/main.css'],
  // ... other configuration
})
```

### Vue 3

If you're using Vue 3, import the CSS file in your main entry file:

```ts [main.ts]
import { createApp } from 'vue'
import App from './App.vue'
import './assets/css/main.css'

createApp(App).mount('#app')
```

## TypeScript Configuration

For better TypeScript support, add path mapping to your `tsconfig.json`:

```json [tsconfig.json]
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./*"],
      "~/*": ["./*"]
    }
  }
}
```
